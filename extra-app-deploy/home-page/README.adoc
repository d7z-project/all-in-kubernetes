== 说明

当集群下内容足够多时，需要一个网站帮你快速跳转，可导入如下配置快速创建一个基于 gitlab pages 的导航站点，依赖 Gitlab Pages 自动渲染内容。

创建完成后，你需要在 `https://gitlab.d7z.net/docs/home` 下创建项目，并生成 Gitlab Page 页面，此导航站将在如下地址提供服务：

* https://d7z.net
* https://home.d7z.net
* https://www.d7z.net

此导航站点依赖如下镜像：

[source,bash]
----
docker.io/library/nginx:1.21.1
----
