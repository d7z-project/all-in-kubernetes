== 基础环境配置

=== 修改 APT 远程仓库地址 (可选)

受地域影响，默认的 APT 下载可能会非常慢，可根据实际需求更改为合适的地址，例如：

[source,bash]
----
include::./script/conf-update-apt-source.sh[lines=2..-1]
----

=== 修改OpenSSH 配置

关闭密码登陆可提高服务器安全性。同时，在后续部署中可能会用到 22 端口，所以将默认 OpenSSH 端口替换成 `2222` ，如果没有使用需求则可以跳过。

CAUTION: 在关闭密码登陆之前请 *务必* 将公钥复制到目标服务器，否则可能会导致无法使用SSH远程登陆。可以使用 `ssh-copy-id <user>@<address>` 快速复制本机的公钥到目标服务器下。

[source,bash]
----
include::./script/conf-update-openssh-server.sh[lines=2..-1]
----
