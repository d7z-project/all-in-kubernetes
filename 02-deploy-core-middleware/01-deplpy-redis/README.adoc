== 测试

当所有机器部署完成后，使用以下目录查看部署结果：

[source%linenums,bash]
----
 kubectl get pvc,secrets,configmaps,pods,service -n core-middle -l type=redis
----

如果一切无误，可使用以下命令测试，正常情况下将会显示 OK 然后退出。

[source%linenums,bash]
----
REDIS_PASSWD=$(kubectl get secrets -n core-middle  secret-redis-root -o jsonpath='{.data.password}' | base64 -d)
kubectl exec -it -n core-middle pods/redis-server-0  -- redis-cli AUTH $REDIS_PASSWD
----
